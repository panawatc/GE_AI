Hardware & Software

เจาะลึกการทำงานของสมองกลและชุดคำสั่ง

Algorithms

เรียนรู้วิธีการแก้ปัญหาที่ฉลาดที่สุด

เลขฐานสอง (Binary System)

ในโลกของมนุษย์ เราใช้เลขฐาน 10 (0-9) เพราะเรามี 10 นิ้ว แต่คอมพิวเตอร์คือวงจรไฟฟ้าที่มีเพียง "เปิด" (1) และ "ปิด" (0) เท่านั้น

ทำไมต้องรู้เรื่องนี้?

เพราะทุกไฟล์ที่คุณส่งทางเน็ต ไม่ว่าจะเป็นรูปภาพ High-def หรือวิดีโอ 4K สุดท้ายจะถูกย่อยเป็นสายธารของเลข 0 และ 1 มหาศาลนั่นเอง

ตัวอย่างการนับเลข:
0 = 0000 | 1 = 0001 | 2 = 0010 | 3 = 0011

Hardware vs Software

ความสัมพันธ์ระหว่างร่างกายและจิตใจของคอมพิวเตอร์

CPU (The Processor)

เหมือนสมองที่คอยคำนวณเลขฐานสองด้วยความเร็วพันล้านครั้งต่อวินาที

RAM (Memory)

โต๊ะทำงานขนาดเล็กที่เอาไว้เก็บของที่กำลังใช้อยู่ ถ้าไฟดับข้อมูลก็หายหมด

Storage (Hard Drive)

ตู้เก็บเอกสารขนาดใหญ่ เก็บทุกอย่างไว้อย่างถาวรแม้ปิดเครื่อง

ขั้นตอนวิธี (Algorithms)

อัลกอริทึมคือ "สูตรอาหาร" ของนักโปรแกรมเมอร์ มันคือขั้นตอนที่บอกว่าต้องทำอะไร 1-2-3 เพื่อให้ได้ผลลัพธ์ที่ต้องการ

หัวใจคือ Efficiency

ในการหาข้อมูล 1 ล้านอย่าง อัลกอริทึมที่ห่วยอาจใช้เวลา 1 ชั่วโมง แต่อัลกอริทึมที่ฉลาด (เช่น Binary Search) อาจใช้เวลาเพียงไม่กี่วินาที

// ตัวอย่าง Algorithm การเช็คตัวเลขคู่-คี่
function checkEvenOdd(number) {
    if (number % 2 == 0) {
        return "เลขคู่";
    } else {
        return "เลขคี่";
    }
}

โครงสร้างข้อมูล (Data Structures)

โครงสร้างข้อมูลคือ "วิธีการจัดเก็บข้อมูล" ให้เหมาะสมกับการใช้งาน เหมือนกับการจัดของในบ้าน ของแต่ละชนิดก็ต้องเก็บคนละที่

Array (อาร์เรย์)

เหมือนตู้ล็อกเกอร์ที่มีหมายเลขกำกับ เข้าถึงข้อมูลได้ทันทีด้วยตัวเลข (index) แต่ขยายขนาดลำบาก

Linked List (ลิงก์ลิสต์)

เหมือนรถไฟที่ตู้แต่ละตู้เชื่อมกัน เพิ่มหรือลบตู้ได้ง่าย แต่ต้องเดินผ่านทีละตู้ถ้าจะหาของ

Stack (สแต็ก)

เหมือนกองจานที่ซ้อนกัน ใส่เข้าสุดท้าย-ออกก่อน (LIFO) เช่น ปุ่ม Undo ในโปรแกรม

Queue (คิว)

เหมือนแถวต่อคิวซื้อของ มาก่อน-ได้ก่อน (FIFO) เช่น ระบบพิมพ์งานของเครื่องปริ้นท์

Tree (ทรี)

เหมือนแผนผังองค์กรหรือต้นไม้ที่แตกกิ่งก้าน เหมาะกับข้อมูลที่มีลำดับชั้น เช่น โฟลเดอร์ในคอมพิวเตอร์

Hash Table (แฮชเทเบิล)

เหมือนพจนานุกรมที่ค้นคำด้วยตัวอักษรนำหน้า เข้าถึงข้อมูลได้เร็วมาก ใช้ใน Dictionary ของ Python

เลือกใช้ให้ถูกกับงาน

ไม่มีโครงสร้างข้อมูลที่ดีที่สุด มีแต่โครงสร้างที่ "เหมาะสม" ที่สุดกับปัญหานั้นๆ การเข้าใจจุดแข็ง-จุดอ่อนของแต่ละแบบจะช่วยให้เขียนโปรแกรมได้มีประสิทธิภาพมากขึ้น

// ตัวอย่าง Array กับ Stack ใน JavaScript
let myArray = [1, 2, 3, 4, 5];
console.log(myArray[2]);  // เข้าถึงตำแหน่งที่ 2 ได้เลย = 3

// Stack ใช้ push และ pop
let stack = [];
stack.push("หน้าแรก");
stack.push("หน้าที่สอง");
stack.push("หน้าที่สาม");
stack.pop();  // ย้อนกลับ = "หน้าที่สาม"

เครือข่ายคอมพิวเตอร์ (Computer Networks)

อินเทอร์เน็ตคือเครือข่ายคอมพิวเตอร์ขนาดใหญ่ที่เชื่อมต่อกันทั่วโลก มาทำความเข้าใจว่าข้อมูลเดินทางอย่างไร

IP Address

เหมือน "บ้านเลขที่" ของคอมพิวเตอร์แต่ละเครื่องบนอินเทอร์เน็ต เช่น 192.168.1.1 ใช้ระบุตำแหน่งปลายทาง

DNS (Domain Name System)

เหมือน "สมุดโทรศัพท์" ที่แปลงชื่อเว็บ (www.google.com) ให้เป็นหมายเลข IP ที่คอมพิวเตอร์เข้าใจ

Router (เราเตอร์)

เหมือน "บุรุษไปรษณีย์" ที่รับข้อมูลแล้วส่งต่อไปยังเส้นทางที่ถูกต้อง จนถึงปลายทาง

HTTP/HTTPS

กฎการสื่อสารของเว็บไซต์ HTTPS เพิ่มการเข้ารหัสข้อมูล ทำให้ปลอดภัยกว่า (สังเกตรูปกุญแจในเบราว์เซอร์)

TCP/IP

เหมือน "ระบบขนส่ง" ที่แบ่งข้อมูลเป็นชิ้นเล็กๆ (packet) ส่งไป แล้วประกอบกลับที่ปลายทาง

Firewall

เหมือน "ยามรักษาความปลอดภัย" ที่คอยตรวจสอบข้อมูลที่เข้า-ออก ป้องกันผู้ไม่หวังดี

การเดินทางของข้อมูลบนอินเทอร์เน็ต

เมื่อคุณพิมพ์ google.com → DNS หาเลข IP → Router ส่งต่อข้อมูล → Server ของ Google รับคำขอ → ส่งหน้าเว็บกลับมา → เบราว์เซอร์แสดงผล ทั้งหมดนี้เกิดขึ้นในเสี้ยววินาที!

พื้นฐานการเขียนโปรแกรม (Programming Basics)

การเขียนโปรแกรมคือการสั่งให้คอมพิวเตอร์ทำงานตามที่เราต้องการ โดยใช้ภาษาที่คอมพิวเตอร์เข้าใจ

ตัวแปร (Variables)

เหมือน "กล่องเก็บของ" ที่มีป้ายชื่อ ใช้เก็บข้อมูลที่เราต้องการใช้งาน เช่น ชื่อ, อายุ, คะแนน

ประเภทข้อมูล (Data Types)

ข้อมูลมีหลายชนิด เช่น ตัวเลขจำนวนเต็ม (int), ทศนิยม (float), ข้อความ (string), จริง/เท็จ (boolean)

เงื่อนไข (Conditions)

คำสั่ง if-else ใช้ตัดสินใจ "ถ้า...ก็..." เช่น ถ้าคะแนนมากกว่า 50 → ผ่าน ไม่งั้น → ไม่ผ่าน

การวนซ้ำ (Loops)

ทำอะไรซ้ำๆ หลายรอบ เช่น for loop, while loop แทนที่จะเขียนโค้ดเดิม 100 ครั้ง

ฟังก์ชัน (Functions)

กลุ่มคำสั่งที่รวมกันไว้ เรียกใช้ซ้ำได้ เหมือน "สูตรสำเร็จรูป" ที่เตรียมไว้ใช้บ่อยๆ

การแก้ไขข้อผิดพลาด (Debugging)

ทุกโปรแกรมมี bug การหาและแก้ไขข้อผิดพลาดเป็นทักษะสำคัญที่โปรแกรมเมอร์ต้องฝึก

// ตัวอย่างพื้นฐานการเขียนโปรแกรม
let name = "สมชาย";      // ตัวแปรเก็บข้อความ
let age = 20;             // ตัวแปรเก็บตัวเลข
let isStudent = true;     // ตัวแปรเก็บค่าจริง/เท็จ

// เงื่อนไข if-else
if (age >= 18) {
    console.log(name + " เป็นผู้ใหญ่แล้ว");
} else {
    console.log(name + " ยังเป็นเด็กอยู่");
}

// การวนซ้ำ for loop
for (let i = 1; i <= 5; i++) {
    console.log("รอบที่ " + i);
}

// ฟังก์ชัน
function greet(personName) {
    return "สวัสดี " + personName + "!";
}
console.log(greet("สมหญิง"));  // "สวัสดี สมหญิง!"

ลอจิกเกต (Logic Gates)

ลอจิกเกตคือ "วงจรพื้นฐาน" ที่ประมวลผลข้อมูล 0 และ 1 เป็นหัวใจของชิป CPU ทุกตัว ประกอบด้วยหลายพันล้านเกตทำงานร่วมกัน

AND Gate (และ)

ได้ 1 ก็ต่อเมื่อ input ทุกตัวเป็น 1
1 AND 1 = 1
1 AND 0 = 0
0 AND 0 = 0

OR Gate (หรือ)

ได้ 1 ถ้ามี input อย่างน้อยหนึ่งตัวเป็น 1
1 OR 1 = 1
1 OR 0 = 1
0 OR 0 = 0

NOT Gate (ไม่)

กลับค่า input
NOT 1 = 0
NOT 0 = 1
เหมือนสวิตช์ไฟที่กลับสถานะ

XOR Gate (เฉพาะหนึ่ง)

ได้ 1 ก็ต่อเมื่อ input ต่างกัน
1 XOR 1 = 0
1 XOR 0 = 1
0 XOR 0 = 0

NAND Gate (ไม่และ)

ตรงข้ามกับ AND
เกตพิเศษที่สามารถสร้างเกตอื่นๆ ทุกชนิดได้!

NOR Gate (ไม่หรือ)

ตรงข้ามกับ OR
เหมือน NAND สามารถสร้างเกตอื่นได้ทั้งหมด

จากเกตสู่คอมพิวเตอร์

ลอจิกเกตหลายตัวรวมกันเป็น Adder (วงจรบวกเลข) → ALU (หน่วยคำนวณ) → CPU (หน่วยประมวลผลกลาง) นี่คือวิธีที่คอมพิวเตอร์ "คิดเลข" ได้จริงๆ!

// จำลองลอจิกเกตด้วย JavaScript
function AND(a, b) { return a && b ? 1 : 0; }
function OR(a, b)  { return a || b ? 1 : 0; }
function NOT(a)    { return a ? 0 : 1; }
function XOR(a, b) { return a !== b ? 1 : 0; }

// ตัวอย่างการใช้งาน
console.log(AND(1, 1));  // 1
console.log(OR(0, 1));   // 1
console.log(NOT(1));     // 0
console.log(XOR(1, 0));  // 1

// Half Adder - วงจรบวกเลขพื้นฐาน
function halfAdder(a, b) {
    let sum = XOR(a, b);    // ผลบวก
    let carry = AND(a, b);  // ตัวทด
    return { sum, carry };
}
